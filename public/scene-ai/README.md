# scene-ai/ — homepage "Scene AI" before/after slider

Two photos drive the drag-to-compare slider in the Scene AI section.

- Format: **.jpg**
- Orientation: **landscape 16:9** (e.g. 1600 × 900). Both should frame the
  **same room from the same angle** so the comparison lines up.
- Shown filling a 16:9 frame (cover).

| Slot   | File         | What it shows                          |
|--------|--------------|----------------------------------------|
| Before | `before.jpg` | the empty / original room              |
| After  | `after.jpg`  | the same room with Sugarwave pieces placed |

Until both files exist, the slider falls back to two shaded tiles
("Your room" / "With Sugarwave") so the drag interaction still works.
