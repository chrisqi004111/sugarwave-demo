# scene-lab/ — 首页 Before/After 范例图

Scene Lab 首页右侧默认展示的 Before/After 拖拽滑块用的两张图。
（客户保存过设计后，首页改为显示其保存的最终图，不再显示这个范例。）

放两个文件，文件名必须完全一致：

| 用途 | 文件名 |
|------|--------|
| Before（空房间 / 清理后场景） | `before.jpg` |
| After（同一房间 + 摆好产品的成片） | `after.jpg` |

## 图片要求

| 项目 | 要求 |
|------|------|
| 关键 | **两张必须是同一个房间、同一视角**，只是 after 多了产品——这样拖动对比才自然 |
| 比例 | **横向 16:9**（两张比例要一致），建议约 1600×900 或更大 |
| 格式 | jpg（带背景即可，不需要透明） |
| 大小 | 建议 < 500KB/张 |

放好后刷新首页即可看到。缺图时自动回退为「拖放上传区」，不会报错。
