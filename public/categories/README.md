# categories/ — homepage "sugarwave Collection" category cards

One photo per product category. Each drives a card in the homepage collection
row; clicking a card opens Shop filtered to that category.

- Format: **.jpg**
- Orientation: **square 1:1** (e.g. 1000 × 1000). Shown filled (cover), so
  keep the subject roughly centered.

Filenames must match each category slug (see `COLLECTIONS` in `src/data.js`):

| Category | File           |
|----------|----------------|
| Seating  | `seating.jpg`  |
| Lighting | `lighting.jpg` |
| Tables   | `tables.jpg`   |
| Objects  | `objects.jpg`  |

Until a file exists, that card shows a shaded tile with the category name.
