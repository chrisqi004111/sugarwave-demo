# Forest — 补充角度图 (supplementary reference photos)

把 **Forest** 产品的多角度照片放进这个文件夹（正面 / 侧面 / 背面 / 细节等，建议 1–4 张）。

- 文件名任意（`front.png`、`side.jpg` 等都行），支持 `png / jpg / jpeg / webp`。
- 放进来即自动生效：Vite 在构建时通过 `import.meta.glob` 收集本目录的图片，
  渲染（TRY → 点位置）时会把这些角度图 **连同产品在 `PRODUCTS` 里写好的尺寸** 一起发给 codex，
  帮助 AI 更准确地理解产品的 3D 形状与比例。
- 这是「持久」的参考图来源；UI 里产品行的 `+ REF` 按钮则是「当次会话」临时追加，二者会合并使用。
## 规格建议

- **比例**：任意。渲染前会保持原比例居中铺到 1024×1024 透明画布（不拉伸变形），所以竖图/横图/方图都行，无需裁成正方形。
- **背景**：纯色或透明最佳，便于 AI 分离出产品本体。
- **格式**：`png / jpg / jpeg / webp`，会统一转成 PNG。
- **命名**：仅用于你自己归档与排序；codex 看到的是 `product1.png/2.png…`，文件名不会传给它。
  想让 codex 知道角度（正/侧/背），需要额外把文件名作为标签写进 prompt（可按需添加）。
