# select/ — 选择模式页（SelectPage）的两张卡片插图

「AI RECOMMENDATION」/「MANUAL SELECTION」选择页，每张卡片上方的配图。

放两个文件，文件名必须完全一致：

| 卡片 | 文件名 |
|------|--------|
| AI RECOMMENDATION（START WITH AI） | `ai-recommendation.jpg` |
| MANUAL SELECTION（START FROM SCRATCH） | `start-from-scratch.jpg` |

## 图片要求

| 项目 | 要求 |
|------|------|
| 比例 | **4:3**（横向），会按 `cover` 裁切填满 |
| 分辨率 | 长边约 800–1200px |
| 格式 | jpg（带背景即可，不需要透明） |
| 大小 | 建议 < 500KB |

放好后刷新页面即可看到。缺图时卡片自动显示灰色占位，不会报错。
