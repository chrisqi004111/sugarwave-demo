# scenes/ — real scene photos (homepage slider)

Vertical (portrait) lifestyle photos of the product styled in a real room.
Used by the big Shop slider on the homepage.

- Format: **.jpg**
- Orientation: **vertical / portrait** (~4:5, e.g. 1000 × 1250)
- Fills the frame edge-to-edge (cover), so keep the subject roughly centered.

Filenames must match each product's `slug` (see `src/data.js`):

| Product            | File              |
|--------------------|-------------------|
| Boop Stool         | `boop.jpg`        |
| Lianlian Lamp      | `lianlian.jpg`    |
| Forest Side Table  | `forest.jpg`      |
| Scratch Vase       | `scratch.jpg`     |
