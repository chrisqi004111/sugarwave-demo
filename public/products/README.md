# products/ — transparent product cut-outs (square line-up)

The product itself on a **transparent background** (PNG cut-out).
Used by the full-width square product line-up below the slider.

- Format: **.png** (transparent background)
- Shown whole, centered on white (contain) — nothing is cropped.
- Square-ish framing works best; some padding around the product is fine.

Filenames must match each product's `slug` (see `src/data.js`):

| Product            | File              |
|--------------------|-------------------|
| Boop Stool         | `boop.png`        |
| Lianlian Lamp      | `lianlian.png`    |
| Forest Side Table  | `forest.png`      |
| Scratch Vase       | `scratch.png`     |
